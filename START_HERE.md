## 📞 Quick Reference

**Where to find things:**
- Components: `src/app/shared/components/`
- Services: `src/app/core/services/`
- Features: `src/app/features/`
- Routes: `src/app/app.routes.ts`
- Styles: Global in `src/styles.scss`, inline with Tailwind

**How to do things:**
- Make HTTP calls: Use `ApiService`
- Handle auth: Use `AuthService`
- Show notifications: Use `NotificationService`
- Store data: Use `StorageService`

**Need to change:**
- API URL: Edit `src/environments/environment.ts`
- Colors: Edit `tailwind.config.js`
- Routes: Edit `src/app/app.routes.ts`
